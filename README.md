Predictor de Tendencias Web App
TrendPredictor es una aplicación web robusta diseñada para el análisis y la predicción de tendencias mediante el procesamiento de datos históricos. Construida bajo el patrón de diseño MVC (Model-View-Controller), la aplicación ofrece una interfaz intuitiva y una lógica de servidor potente para ayudar a los usuarios a anticipar movimientos en el mercado o datos estadísticos.

-Características Principales
Motor de Predicción: Algoritmos implementados en C# para el análisis de patrones.

Interfaz Dinámica: Vistas interactivas desarrolladas con Razor Pages.

Arquitectura Limpia: Separación estricta de responsabilidades mediante el patrón MVC.

Gestión de Datos: Soporte para entrada de datos masivos y visualización de resultados en tiempo real.

-Stack Tecnológico
Lenguaje: C#

Framework: .NET 8.0 

Patrón de Arquitectura: ASP.NET Core MVC

Frontend: Razor, HTML5, CSS3, BootStrap 5

ORM: Entity Framework Core
